Fuel Efficiency

Customers are very concscious with fuel efficiency, for various reasons:

  • rising cost of fuel
  • increasing concern for the environment

It's important to be able to predict the fuel efficiency of new cars during the design stage. Many factors can effect the fuel efficiency, including:

  • number of cylinders
  • horsepower
  • weight
  • engine displacement
  • rear axle ratio
  • transmission type

prediction model

Using a linear model, I determined the effect of weight, number of cylinders, and horsepower on fuel efficiency (mpg), and used this model to create a function to predict fuel efficiency based on these factors.

fit <- lm(mpg ~ wt + cyl + hp, data=mtcars)
fit$coefficients

## (Intercept)          wt         cyl          hp 
##  38.7517874  -3.1669731  -0.9416168  -0.0180381

fitted_mpg <- function(wt, cyl, hp) {
  fit$coefficients[1] + fit$coefficients[2] * wt + 
    fit$coefficients[3] * cyl + fit$coefficients[4] * hp
}
